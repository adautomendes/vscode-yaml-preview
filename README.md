# vscode-yaml-preview

Documents written in YAML format can be previewed in real time on VS Code as a table.

![preview](./preview.png)

## How to use

- Open any `YAML` file.
- Type `CTRL+SHIFT+P` and search for 'yaml'.
- Choose either `Yaml:Open preview` or `Yaml:Open preview to the side`
